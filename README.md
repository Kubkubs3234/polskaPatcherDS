# polskaPatcherDS  
  
Patcher jęz. polskiego dla NDS  
WYMAGANIA:  
PC z ubuntu / Bootable USB / ewentualnie WSL  
Karta R4   
Czytnik SD
SD  
Coś do OSTROŻNEGO zwarcia SL1  
  
INSTRUKCJA:  
  
1. Wbootuj do live USB ubuntu, przejdź fragment setupu i wybierz "try ubuntu".  
Internet nie jest konieczny.  
2. Pobierz mój pakiet narzędzi (code > pobierz ZIP)  
3. Wypakowany folder daj na pulpit  
4. Wypakowany folder kliknij Prawym Przyciskiem Myszy > open in Terminal  
5. Pisz w terminalu:  
   sudo ./polskapatcher  
6. Program poda dokładne instrukcje.  
   UPEWNIJ SIE, ŻE MASZ BACKUP FIRMA I FLASHME WGRANE PRZED ZMIANACH W DS-OWYM FLASHU!!!
